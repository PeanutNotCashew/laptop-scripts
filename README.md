# Scripts
A repository of miscellaneous scripts for my laptop

## Status scripts
The following scripts in this category use notifications to display information to users. Therefore, a notification daemon is required.

### notify-battery
A script to send a notification when battery is getting low, or getting high. Additionally, hibernates when the battery is too low. I run this with a cron job.

### time-since-last-update
A script to send a notification describing the last time the Gentoo repositories were synced. This generally describes the time that packages were updated. I run this in my Openbox autostart file.
